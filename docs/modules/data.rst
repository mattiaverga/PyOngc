Using data with Pandas
======================

The `data` module provides some function for using OpenNGC data
with `Pandas <https://pandas.pydata.org/>`_. Pandas is a powerful
Python tool for data analysis and manipulation. OpenNGC data is
provided as Pandas' DataFrames.
